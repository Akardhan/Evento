Our Mission

Our mission is to provide a unique platform for user looking to host a event list the places for hosting an specific event at that location using Google API. Also after logging in the website user gets a dashboard specific to his/her account. The data gets saved in our database.

Our Plan

It is our plan to make the world a better place by saving time for the users. We will use API, HTML , JavaScript , CSS, Bootstrap to create a very user friendly environment. An effective platform for every user. With our great team we will try to make this project among the finest.

Our Vision

To provide the user will not have to move to different websites making task of event Hosting hectic and difficult. He/ She can work and keep the track using his/her dashboard. Thus saving user precious time and energy.
